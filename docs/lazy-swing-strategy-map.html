1. Overview

LazySwing is a 30m-resampled, always-in-market trend-follower built on Supertrend (ATR 25 / mult 1.75). The base behaviour is simple: when Supertrend flips, exit the current side and enter the opposite side on the next bar. Around this core, a sequence of gates filter or modify the flip and a sequence of early-exit mechanisms can leave the trade before the next ST flip.

The strategy is evaluated on every 5-minute bar. Most decisions are deferred to hourly closes (resampled 30m bars), but a few mechanisms (fast_exit, regime_trail_stop, held_flip safety) can fire on any 5m bar.

Reading order: start with the state lifecycle for the macro picture, then drop into the per-state flow charts to see the exact evaluation order for each gate. The exit pipeline is the single most important diagram — order matters.

3. Per-bar exit pipeline — LONG & SHORT (precedence)

On every 5m bar, while in a position (LONG or SHORT), the code evaluates these checks in this order and returns on the first one that fires. This is the most-asked question when reasoning about the strategy: "Why did fast_exit fire before the ST flip resolved?" — because fast_exit is earlier in the chain.

Diagram describes both sides. For SHORT, mirror the conditions: fast_exit fires when price is above ST, held_flip arms direction "long", and gain = (entry / close − 1). Exit actions are COVER instead of SELL.

9. Indicator glossary

IndicatorWhat it measuresWhere it's used
Supertrend (ST) ATR-based trend line; price above = bullish, below = bearish. Flips when price crosses. Core flip signal. Both primary and a confirmation series.
flip_vol_ratio 4-bar short vol / 336-bar long mean vol on resampled bars. Ratio ≥ active_min = "real" flip. Gates every ST flip. Active min interpolated by vol regime.
Vol regime weight 0–1 score from a slower vol-ratio comparison. Drives "squared" interpolation between low/high parameter pairs. Modulates flip_vol_ratio min, fast_exit RVOL min, held_flip stop pct.
5m RVOL 24-bar short / 2016-bar long realised-vol ratio on 5m bars. Burst detector. Gates fast_exit firing.
ADX (avg directional index) Trend strength 0–100. ≥ 20 = real trend; rising = strengthening. regime_trail prerequisite (need ≥ 20 prior) + drop test (fade ≥ 2.5%).
KC z-score / BB z-score How many σ price is outside Keltner/Bollinger bands. High z = "stretched" / over-extended. regime_trail stretch test. KC ≥ 1.75 OR BB ≥ 2.75 = stretched.
Kaufman Efficiency Ratio (ER) |close[end] − close[start]| / Σ|Δclose|. 0 = pure chop, 1 = perfect trend. ER gate (48 5m bars ≥ 0.32 = trend → bypass vol_ratio rejection).
HMACD (Heikin-MACD) MACD computed on Heikin-Ashi-smoothed prices, periods 24/51/12 on resampled bars. Used by some older modes for confirmation; not load-bearing in current config.
Stretch lookback Whether KC/BB z thresholds were breached in the last 3 resampled bars. regime_trail stretch test (must be recently stretched, not just at peak).
